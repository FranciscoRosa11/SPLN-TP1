Instruções para a execução do programa:

1. Correr o script run.sh (este script vai buscar todas as páginas HTML das várias pessoas após recolher os ids a partir da página principal)
2. Correr program.py da seguinte forma: python3 program.py 'ID.html' (a partir de ids.txt escolher um id e substituir ID com o id que se quer procurar)
